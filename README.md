# Reno Notice Board

A full-stack Notice Board application built as part of a technical assignment.

## Live Demo

https://reno-notice-board-three.vercel.app/

## GitHub Repository

https://github.com/yasir78604/reno_notice_board.git

## Tech Stack

* Next.js
* Prisma ORM
* TiDB Cloud Database
* Tailwind CSS
* Axios
* Vercel Deployment

## Features Implemented

* Create new notice
* View all notices
* Edit existing notice
* Delete notice
* Delete confirmation modal
* Responsive user interface
* API Routes using Next.js
* Server-side validation
* Urgent notices displayed first based on priority

## Project Structure

/pages
/components
/lib
/prisma
/styles

## Installation

Clone the repository:

npm install

Run locally:

npm run dev

## Deployment

Deployed using Vercel with environment variables configured.

## Future Improvements

* Image upload support
* Search and filter functionality
* Better notification system
